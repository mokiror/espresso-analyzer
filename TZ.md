# Техническое задание для проекта: Espresso Analyzer

## 1. Цель проекта
Создание веб-сайта для анализа кофе. Система помогает любителям кофе подобрать оптимальные параметры заваривания (температуру воды и помол) на основе характеристик зёрен, обеспечивая сбалансированный вкус напитка в домашних условиях.

## 2. Роли пользователей

**Гость:**
- Просмотр каталога сортов кофе
- Просмотр рецептов 
- Детальный просмотр прогнозов вкуса с графиками

**Администратор (через Django Admin):**
- Управление сортами кофе
- Управление рецептами заваривания
- Управление прогнозами вкуса
- Назначение рекомендуемых рецептов

## 3. Модели данных (сущности)

### Модель 1: CoffeeBean (Сорт кофе)
Хранит информацию о кофейных зёрнах:
- `name` — название сорта
- `origin_country` — страна происхождения
- `region` — регион выращивания
- `altitude_min` / `altitude_max` — высота произрастания 
- `process_method` — метод обработки 
- `acidity_score` — оценка кислотности 
- `density` — плотность зерен
- `description` — текстовое описание

### Модель 2: BrewRecipe (Рецепт заваривания)
Хранит параметры заваривания для конкретного сорта:
- `coffee` — связь с CoffeeBean
- `water_temperature` — температура воды
- `grind_size` — степень помола
- `brew_time_seconds` — время заваривания
- `coffee_dose` — доза кофе
- `water_volume` — объём воды
- `is_recommended` — рекомендуется или нет

### Модель 3: TastePrediction (Прогноз вкуса)
Хранит результаты мат расчёта:
- `recipe` — связь с BrewRecipe 
- `acidity_intensity` — интенсивность кислотности 
- `sweetness_intensity` — интенсивность сладости 
- `bitterness_intensity` — интенсивность горечи 
- `balance_type` — тип 
- `recommended_temperature_adjustment` — рекомендация по температуре
- `tds_estimate` — процент растворённых веществ
- `extraction_yield` — процент экстракции

## 4. Ключевой функционал (User Stories)

1. **Просмотр каталога сортов кофе**
   - Пользователь заходит на страницу со списком сортов
   - Может искать по названию или стране происхождения
   - Видит основные характеристики каждого сорта

2. **Детальный просмотр сорта кофе**
   - Пользователь выбирает сорт из списка
   - Видит полную информацию о сорте
   - Видит все рецепты для этого сорта

3. **Просмотр рецепта и прогноза вкуса**
   - Пользователь открывает страницу рецепта
   - Видит параметры заваривания
   - Видит индикатор баланса
   - Видит график зависимости вкуса от температуры

4. **Управление данными через админку**
   - Администратор добавляет новые сорта кофе
   - Администратор создаёт рецепты для сортов
   - Система автоматически рассчитывает 
   - Администратор отмечает рекомендуемые рецепты
