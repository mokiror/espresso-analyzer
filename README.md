# Espresso Analyzer — Система анализа кофе

Веб-сервис для любителей кофе, помогающий подобрать оптимальные параметры заваривания на основе характеристик зёрен. Система использует математические модели для прогнозирования вкусового профиля и показывает результаты с помощью Plotly.

## С помозщью чего делали

- **Python** 3.14
- **Django** 6.0.6
- **Plotly** 5.18.0 — интерактивная визуализация
- **Pandas** — обработка данных
- **Bootstrap** 5.3 — адаптивный интерфейс
- **SQLite** — база данных (разработка)
